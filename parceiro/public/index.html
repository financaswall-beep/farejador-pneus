Localização base

O plano B de quando o cliente não manda a localização. Quem decide a entrega é o raio em km acima.

Como funciona

Só entrega, só retirada, ou os dois — você decide.
O raio de entrega é em km: a Rede só te manda entrega dentro dele. Sem raio = só retirada.
A retirada não tem limite de distância — quem vem buscar, vem de onde quiser.
A cidade base é o plano B de localização — não limita a entrega, quem limita é o raio.
